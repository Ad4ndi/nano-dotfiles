# nano-dotfiles
Dotfiles for i3wm/polybar

## Dependencies
### Main
- i3
- polybar
- alacritty
- rofi
- maim
- xclip

### Optional
- helix
- fish

## Setup
```bash
git clone https://github.com/Ad4ndi/nano-dotfiles
cp -r nano-dotfiles/.config/* ~/.config
rm -rf nano-dotfiles
```
